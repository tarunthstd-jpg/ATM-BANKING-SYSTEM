# 💳 SmartBank ATM System (Python)

## 📌 Description

This is a simple **ATM simulation program** written in Python. It allows users to securely access their account using a PIN and perform basic banking operations like checking balance, depositing, withdrawing money, and viewing transaction history.

---

## 🚀 Features

* 🔐 PIN authentication (3 attempts)
* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money (with balance check)
* 📜 View transaction history
* 🔁 Menu-driven interface

---

## 🛠️ Technologies Used

* Python (Core Concepts)

  * Functions
  * Loops
  * Conditional statements
  * Lists

---

## ▶️ How to Run

1. Install Python on your system
2. Save the file as `atm.py`
3. Open terminal or command prompt
4. Run the program:

   ```bash
   python atm.py
   ```

---

## 🔑 Default Details

* PIN: `1234`
* Initial Balance: `₹15000`

## 📌 Notes

* Maximum 3 incorrect PIN attempts allowed
* Program exits if all attempts are used
* Transaction history is stored during runtime only
